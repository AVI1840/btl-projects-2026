# Product Overview

This is the **AI Projects Portfolio** for the Benefits Administration (מינהל גמלאות) at Israel's National Insurance Institute (ביטוח לאומי).

It serves as a showcase and documentation hub for AI-powered digital tools that improve citizen benefit utilization and public service.

## Key Facts
- All content is in Hebrew (RTL)
- The portfolio tracks ~16 AI projects, ~9 live in production
- Serves ~10M citizens across 98 benefit types
- Tools are hosted on GitHub Pages (avi1840.github.io) and on an internal intranet (snifim.blroot)
- Each tool has a "one-pager" HTML summary page and a Word (.docx) document

## Audience
- Internal stakeholders and decision-makers at ביטוח לאומי
- Field workers (פקידי שירות) who use the tools with citizens
- The portfolio itself is public-facing via GitHub Pages
